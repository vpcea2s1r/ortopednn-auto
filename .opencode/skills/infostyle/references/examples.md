## Формы: лейблы и подсказки

**До:**
```
Введите ваш электронный адрес *
[Поле]
Это обязательное поле
```

**После:**
```
Email
[ivan@mail.ru]
Отправим результат на этот адрес
```

**Почему:** лейбл краткий, placeholder показывает формат, подсказка объясняет зачем.

## Лендинг: hero-секция

**До:**
```
Уникальный AI-сервис нового поколения
Наша инновационная платформа использует передовые технологии
искусственного интеллекта для создания качественных презентаций
[Попробовать бесплатно]
```

**После:**
```
Презентация за 5 минут
Загрузите тему — получите готовые слайды с текстом и оформлением.
15 000 студентов уже сдали работы.
[Создать презентацию бесплатно]
```

**Что изменилось:**
1. «Уникальный AI-сервис нового поколения» → конкретная выгода с цифрой
2. «Инновационная платформа использует передовые технологии» → что именно делает
3. Добавлено социальное доказательство с конкретной цифрой
4. CTA: действие + что получит + «бесплатно» снимает риск

## Форма обратной связи

**До:**
```
Что не подходит?

🟢 Дорого для меня
🟢 Не устраивает предпросмотр работы
    - Страниц (или слайдов) больше/меньше, чем нужно
    - Оформление не подходит
    - Учитель дал особые требования
    - Нужна другая практическая часть (для проекта)
🟢 Боюсь, что преподаватель "спалит"
    - Боюсь, что такая же работа будет у другого
    - Не уверен, что пройдёт антиплагиат
🟢 Дедлайн ещё далеко, сделаю потом
🟢 Просто смотрю, что умеет сервис
🟢 Хочу попробовать сам
🟢 Другое — напишите в поле ниже
```

**После:**
```
Почему не оформили заказ?

○ Цена не устраивает
○ Не то количество страниц или слайдов
○ Не подходит оформление
○ Есть особые требования от преподавателя
○ Нужна другая практическая часть
○ Переживаю за уникальность работы
○ Дедлайн ещё нескоро
○ Пока изучаю сервис
○ Попробую сделать сам
○ Другое
```

**Что изменилось:**
1. Вопрос конкретный: «Почему не оформили заказ?» вместо «Что не подходит?»
2. Плоский список — проще сканировать и тыкнуть
3. «Боюсь, что спалит» → «Переживаю за уникальность» — точнее, без сленга
4. Убраны подпункты — каждый вариант самостоятелен
5. «Напишите в поле ниже» — убран канцелярит, и так понятно

## Модальное окно подтверждения

**До:**
```
Вы уверены?
Вы действительно хотите удалить этот элемент?
Данное действие нельзя будет отменить.
[Ок] [Отмена]
```

**После:**
```
Удалить проект «Физика 9 класс»?
Проект, файлы и история генераций будут удалены.
Восстановить нельзя.
[Удалить проект] [Оставить]
```

**Что изменилось:**
1. Заголовок: конкретное действие с названием объекта
2. Тело: что именно удалится
3. Кнопки: конкретные действия вместо «Ок/Отмена»

## Коммерческое описание

**До:**
```
Наша компания является лидирующим поставщиком качественных
образовательных решений. Команда высококвалифицированных
профессионалов с индивидуальным подходом к каждому клиенту
обеспечивает высокий уровень сервиса.
```

**После:**
```
Делаем презентации и проекты для студентов.
3 формата: PowerPoint, PDF, Word.
Среднее время — 5 минут.
15 000 работ сдано с оценкой 4 и выше.
```

**Что изменилось:**
1. «Лидирующий поставщик» → убран корпоративный штамп
2. «Качественных образовательных решений» → что конкретно делают
3. «Команда профессионалов с индивидуальным подходом» → цифры и факты
4. Каждое предложение несёт новую информацию
