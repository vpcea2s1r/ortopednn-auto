# Blog Content Standards

## Объём контента

- **Минимум:** 7000 знаков (950+ слов) видимого текста для статей по высокочастотным запросам
- **Минимум:** 5000 знаков (700+ слов) для низкочастотных запросов
- **Оптимум:** 9500–11000 знаков (1300–1500 слов) — для большинства статей блога
- **Lead-абзац:** 1–2 предложения, которые отвечают на главный вопрос статьи

> Объём считается по видимому тексту (без HTML-тегов, frontmatter, CSS, компонентов).

## Обязательные секции

1. **Lead-абзац** — краткий ответ на ключевой вопрос пациента
2. **Основное содержание** — 4–6 h2-разделов с h3 подразделами (где уместно)
3. **FAQ** — минимум 5 вопросов, максимум 7–8
4. **Источники** — 1–2 ссылки на PubMed, Journal of Prosthetic Dentistry и т.п.
5. **CTA-блок** — призыв к консультации с номером телефона

## Дополнительные секции (где уместно)

- **Сравнительная таблица** — 2+ варианта, 5+ критериев
- **Таблица материалов** — прочность, эстетика, совместимость
- **Таблица этапов** — последовательность процедуры
- **Преимущества / Недостатки** — списки по 4–5 пунктов
- **Осложнения** — возможные риски и их решение
- **Уход** — рекомендации по гигиене и эксплуатации

## Структура статьи

```
h1 — заголовок (с ключевым запросом)
  lead-абзац
  h2 — основной раздел
    h3 — подраздел (если нужно)
    текст, таблицы, списки
  h2 — следующий раздел
  ...
  h2 — FAQ (Часто задаваемые вопросы)
  RelatedArticles
  CTA
```

## Авторство

- Все статьи подписаны: «Никитина М.Г., стоматолог-ортопед»
- Дата публикации и обновления
- JSON-LD Article schema с author/@id

## Мета-теги

- **title:** `{Ключевой запрос} | Никитина М.Г.` (макс. 70 символов)
- **description:** 150–170 символов, содержит ключевой запрос
- **breadcrumbTitle:** краткий (3–5 слов)

## Ссылки и источники

- Минимум 1 источник: PubMed / Journal of Prosthetic Dentistry / International Journal of Prosthodontics / Cochrane review
- Источники не старше 10 лет (предпочтительно 2016–2026)
- Цитата в тексте с названием журнала и годом

## Запреты (SEO)

- Цены не указывать (см. SEO.md)
- Никаких «запишитесь онлайн» — только телефон
- Noindex-страницы: `/preview/`
- Изображения: не добавлять без явного запроса

## Проверка перед публикацией

1. `npm run build` — 0 ошибок
2. Контент: проверить длину (7000+ знаков)
3. Секции: все обязательные присутствуют
4. Источники: реальные PubMed/JD/Cochrane (не выдуманные)
5. Авторство: дата, имя врача
