# Yandex.Dzen — канал стоматолога-ортопеда

## Описание

Канал на Яндекс.Дзен для автоматической публикации статей с ortopednn.ru. Контент подаётся через RSS-ленту сайта, Dzen самостоятельно импортирует новые записи.

## Текущий статус

- **RSS-лента**: `/rss.xml` — создана, работает (448KB, 82+ статьи)
- **Канал Дзен**: зарегистрирован, [dzen.ru/ortopednn](https://dzen.ru/ortopednn)
- **Верификация**: пройдена (meta tag `<meta name="zen-verification" content="GG06PtAHU8BI5KPxbsNGDi4NV4mgleoKANTEXBv7ihSuq6L1BuWBWguS7JP9JXzS"/>`)
- **RSS-импорт**: не подключён (нужно добавить источник `https://ortopednn.ru/rss.xml` в настройках канала)
- **Ссылка на сайте**: добавлена в подвал всех страниц (commit `6182e9ff`, 03.06.2026)
- **Публикация**: автоматическая через RSS после подключения (требуется 10 подписчиков)

## Схема работы

```
Статья в data/blog-articles.ts
    → npm run build
    → /rss.xml обновляется
    → Dzen (RSS-импорт) → автоматическая публикация
```

## Настройка RSS для Дзен

1. Зарегистрировать канал на [dzen.ru](https://dzen.ru)
2. В настройках канала добавить источник: `https://ortopednn.ru/rss.xml`
3. Дзен сам проверяет ленту и публикует новые статьи

### Текущее состояние RSS

- `<title>` — ✅ есть
- `<description>` — ✅ есть
- `<pubDate>` — ✅ есть
- `<link>` — ✅ есть
- `<language>ru</language>` — ✅ есть
- `<content:encoded>` — ✅ полный текст статьи (CDATA с HTML)
- `<enclosure>` — ✅ изображение (og-image.svg)
- `<category>` — ✅ категория статьи
- `<guid isPermaLink="true">` — ✅ уникальный URL

Файл RSS: `src/pages/rss.xml.ts` — полностью готов к Дзен

## Контент-стратегия для Дзен

- Те же статьи, что в блоге ortopednn.ru (RSS один)
- Формат: полезные статьи по стоматологии для пациентов
- Не публиковать: цены, прямые продажи, рекламные тексты (уже удалены с сайта)
- Частота: 1-2 статьи в неделю (совпадает с автопайплайном)

## Telegram-канал (@ortopednn)

### Текущий статус
- **Канал**: создан `@ortopednn`
- **Бот-админ**: `@ortopednn52_bot` — ✅ добавлен администратором (02.06.2026)
- **Авто-постинг**: реализован в `server/bot.js` — функция `postToChannel()`
- **Триггеры**: при генерации новой статьи через `/autogen` или `/horizon` pipeline

### Схема работы

```
Pipeline генерирует статью → сохраняет черновик → postToChannel()
                                                            ↓
                                              Telegram-канал @ortopednn
                                              «📄 Название статьи
                                               Читать: ortopednn.ru/blog/slug/»
```

### Настройка

1. Создать канал (✅ сделано)
2. Добавить `@ortopednn52_bot` администратором канала
3. Убедиться, что `TELEGRAM_CHANNEL` указан в `.env` (по умолчанию `@ortopednn`)
4. Пересобрать контейнер бота
