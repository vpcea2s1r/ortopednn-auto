# Yandex.Dzen — канал стоматолога-ортопеда

## Описание

Канал на Яндекс.Дзен для автоматической публикации статей с ortopednn.ru. Контент подаётся через RSS-ленту сайта, Dzen самостоятельно импортирует новые записи.

## Текущий статус

- **RSS-лента**: `/rss.xml` — создана, работает (42KB, все 62+ статьи)
- **Канал Дзен**: не зарегистрирован (нужен аккаунт пользователя)
- **Публикация**: автоматическая через RSS после регистрации

## Схема работы

```
Статья в data/blog-articles.ts
    → npm run build
    → /rss.xml обновляется
    → Dzen (RSS-импорт) → автоматическая публикация
```

## Настройка RSS для Дзен

1. Зарегистрировать канал на [dzen.ru](https://dzen.ru)
2. В настройках канала добавить источник: `https://ortopednn.ru/rss.xml`
3. Дзен сам проверяет ленту и публикует новые статьи

### Требования Дзен к RSS

- `<title>` — ✅ есть
- `<description>` — ✅ есть
- `<pubDate>` — ✅ есть (RFC 2822)
- `<link>` — ✅ есть
- `<language>ru</language>` — ✅ есть
- Полный текст статьи — ❌ отсутствует (только description)
- Изображения — ❌ отсутствуют

### Улучшения RSS (опционально)

Если Дзен будет показывать превью без контента, можно добавить:

- **Полный текст статьи** в `<content:encoded>` (CDATA с HTML)
- **Изображение** через `<enclosure>` (первое изображение статьи)
- **Категории** через `<category>`

Файл: `src/pages/rss.xml.ts`

## Контент-стратегия для Дзен

- Те же статьи, что в блоге ortopednn.ru (RSS один)
- Формат: полезные статьи по стоматологии для пациентов
- Не публиковать: цены, прямые продажи, рекламные тексты (уже удалены с сайта)
- Частота: 1-2 статьи в неделю (совпадает с автопайплайном)

## Результат

- Дополнительный трафик на ortopednn.ru
- Прямые ссылки в статьях Дзен → на сайт
- Улучшение поведенческих факторов (косвенно)
