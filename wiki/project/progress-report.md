## Рекомендации по Core Web Vitals

### TBT (>200ms на mobile)
- **Lazy loading изображений** — проверить что все изображения ниже сгиба имеют loading="lazy"
- **Убрать render-blocking ресурсы** — JS/CSS, не используемые на первой закраске
- **Оптимизация шрифтов** — Inter подключён через Fonts API, проверить display=swap
- **Code splitting** — Astro уже делает это, но проверить bundle-анализ

### CLS (0.14 на услугах, >0.1)
- **Фиксированные размеры изображений** — width/height у всех img не в SVG
- **Резервирование места под JSON-LD** — может сдвигать контент
- **Отложенная загрузка шрифтов** — проверить нет ли shift при подгрузке Inter

### Best Practices (77 везде)
- **iframe title** — добавить title к iframe (Яндекс.Метрика)
- **Third-party cookies** — ожидаемо для Метрики, некритично
