## Как проверить пересечение с сайтом (Intersection Check)

Любой новый запрос проверяется так перед Wordstat:

```
grep -i "ключевая фраза" src/pages/ --include="*.astro" -r
```

Если есть совпадения — открыть файл, оценить глубину покрытия. Если нет — бить в Wordstat.

## Рекомендация

Приоритет Wordstat-проверки: 1→2→3→4→5 (остальные — если есть время).
Писать статьи по темам с Wordstat >0 (из проверенных — "протезирование при беременности" (1) и "временный протез сколько носить" (2)).
Для непроверенных тем — сначала получить API-ключ AI Studio для Wordstat API.
