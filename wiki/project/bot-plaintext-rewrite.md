# Bot: Plain Text Rewrite (без PubMed)

## Проблема
При пересылке сообщения (plain text) бот искал статьи в PubMed и использовал ПЕРВЫЙ результат рерайта, даже если он был нерелевантен.

Реальный случай: пользователь отправил «Шкала Vita — ориентир для подбора цвета зубов» → PubMed вернул статью про антидепрессанты → бот создал черновик «Влияние воспаления на метаболизм психотропных препаратов».

## Решение
Удалён вызов `searchPubMed()` из обработчика plain text (`server/bot.js` строка 821).
Теперь forwarded message → `rewrite(null, text)` — рерайт текста напрямую, без PubMed.

## Файлы
- `server/bot.js:815-830` — обработчик plain text

## PubMed остаётся
- `/research <тема>` — поиск PubMed с выбором статьи
